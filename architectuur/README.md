# BRP Gebeurtenissen en Notificaties Architectuur

De [C4 modellen](https://c4model.com) zijn gebruikt om de architectuur van BRP Gebeurtenissen en Notificaties te visualiseren. De volgende C4 modellen zijn gebruikt:
- System context diagram
- Container diagram

## System Context diagram
De System Context diagram geeft een hoog overview van een software systeem (de BRP Gebeurtenissen en Notificaties) en de direct verbonden externe systemen (de BRP database en systemen van externe afnemers).

## Container diagram
De Container diagram geeft een hoog overview van de applicaties en data stores waaruit het software systeem bestaat.
