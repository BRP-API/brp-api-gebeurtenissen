# BRP API Gebeurtenissen OpenApi specificatie

## Reviewen van een geresolved OpenAPI specificatie gedurende de API design fase

- Navigeer naar de branch waar de OpenAPI specificatie staat, bijv. de [chore(mutatie-service)38-oas-opgave-van-verhuizing](https://github.com/BRP-API/brp-api-gebeurtenissen/tree/chore(mutatie-service)38-oas-opgave-van-verhuizing)
- Navigeer naar de geresolved OAS bestand onder de specificaties map, bijv. het OAS bestand van de [mutatie service](https://github.com/BRP-API/brp-api-gebeurtenissen/blob/chore(mutatie-service)38-oas-opgave-van-verhuizing/specificaties/mutatie-service/resolved/openapi.yaml)
- Klik op de `Raw` knop om het OAS bestand zonder styling te openen, bijv. het OAS bestand van de [mutatie service](https://raw.githubusercontent.com/BRP-API/brp-api-gebeurtenissen/refs/heads/chore(mutatie-service)38-oas-opgave-van-verhuizing/specificaties/mutatie-service/resolved/openapi.yaml?token=GHSAT0AAAAAADAMHFWMNADZDZX6FC2SY6S22HQZ3FA)
- kopieer de url
- Navigeer naar de [Redoc viewer](https://redocly.github.io/redoc)
- plak de url in de `URL to a spec to try` textbox en klik op de `TRY IT` knop
