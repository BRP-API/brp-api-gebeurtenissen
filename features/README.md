# BRP Gebeurtenissen en Notificaties Functionaliteit

De functionaliteit van BRP Gebeurtenissen en Notificaties is beschreven met voorbeelden. Hiervoor is [Gherkin] gebruikt. De voorbeelden/scenarios zijn georganiseerd in feature bestanden in de volgende mappen:
- user-journeys. Één feature bestand in deze map beschrijft end to end een (deel van een) functionaliteit met als doel om de lezer een beeld te geven van de verschillende taken die binnen BRP Gebeurtenissen en Notificaties hebben plaatsgevonden. Het gedrag van elke taak is beschreven in een eigen scenario om de verantwoordelijkheden van elke taak te benoemen
